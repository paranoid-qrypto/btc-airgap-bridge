## Dependencies & Verification

This tool is designed to be a secure, self-contained HTML file with no external dependencies. The following third-party libraries have been downloaded and included locally in the `js/` directory.

To ensure transparency and allow for independent verification, the exact versions, source links, and SHA-256 file hashes are provided below.
